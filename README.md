
## Description

This is a POC for a vacancy in [Makeen](https://www.makeen.io/). It's my first [Nest](https://github.com/nestjs/nest) project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
